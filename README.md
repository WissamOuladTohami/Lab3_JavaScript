# 👤 User Management System (Express + PostgreSQL)

A simple full-stack CRUD application built with **Node.js, Express, PostgreSQL, and EJS**.  
It allows you to create, update, delete, and display users in a clean web interface.

---

## 🚀 Features

- ➕ Add new users
- ✏️ Edit existing users
- 🗑️ Delete users
- 📋 Display users list
- ⚡ Real-time server with nodemon
- 🛡️ Input validation with express-validator
- 🎨 Clean EJS frontend UI

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- PostgreSQL
- EJS (Embedded JavaScript Templates)
- express-validator
- dotenv
- nodemon
